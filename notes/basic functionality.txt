Real Example



Jab browser ya frontend server ko data bhejta hai, data different formats me ho sakta hai.

Example frontend ye bhej raha hai:

{
  "name": "Mozam",
  "age": 21
}

Ye JSON format hai.

Lekin Express by default is JSON data ko samajh nahi pata. Agar tum bina middleware ke check karo:

app.post('/user', (req, res) => {
    console.log(req.body);
});

to req.body aksar:

undefined

aa sakta hai.

Isliye hum middleware lagate hain:

app.use(express.json());

Ye middleware request ke andar aane wale JSON data ko read karta hai aur usko JavaScript object me convert karke req.body me rakh deta hai.
